## Calculadora web

Pequeña aplicación de calculadora (HTML/CSS/JS) incluida en este repositorio.

- Abrir el archivo [index.html](index.html) en un navegador.
- O servir la carpeta localmente con Python y abrir http://localhost:8000:

```bash
python3 -m http.server 8000
# luego abrir http://localhost:8000
```

Archivos principales:

- [index.html](index.html)
- [styles.css](styles.css)
- [app.js](app.js)
